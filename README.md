# Project-SALT-E
Maturitätsprojekt SALT-E

In diesem Repository sind alle Programme enthalten, die während der Entwicklung des Roboters geschrieben wurden, sowie alle endgültigen Programme zum Betreiben des Roboters.

Die Programme die zum Betreiben des Roboters nötig sind befinden sich auf der obersten Ebene. Jegliche Programme in den Unterordnern sind Programme, die während der Entwicklung und zum Testen geschrieben wurden.
